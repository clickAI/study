# AI 에이전트로 웹서비스 만들기 — 스터디 그룹

이 저장소는 사내 스터디 그룹 발표자료를 호스팅합니다.

## 1주차 — Introduction

GitHub Pages: **https://clickai.github.io/study/**

- `index.html` — 1주차 발표자료 (단일 파일, 오프라인 작동)
- 슬라이드 이동: `←` / `→` 키, 또는 화면 클릭
- PDF 저장: 브라우저에서 `⌘P` / `Ctrl+P`

## GitHub Pages 활성화 방법

1. 이 저장소의 **Settings → Pages**로 이동
2. **Source**를 `Deploy from a branch`로 선택
3. **Branch**: `main` / **Folder**: `/ (root)` 선택 후 Save
4. 1~2분 후 위 URL에서 접속 가능

> `.nojekyll` 파일이 포함되어 있어 Jekyll 처리 없이 그대로 서빙됩니다.
